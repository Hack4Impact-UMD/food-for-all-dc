# Calendar Page

Interactive calendar for viewing and managing deliveries. Supports month and day views with recurring delivery patterns.

## Key Features

- Month/Day view switching
- Recurring delivery pattern calculation
- Delivery limits per date
- Event creation and editing
- Integration with DayPilot calendar library

## Components

- `CalendarPage.tsx` - Main page component
- `components/` - Calendar-specific components (views, dialogs, utilities)
- Uses `RecurringDeliveryContext` for recurring delivery state
