# Delivery Management Page

Delivery assignment, clustering, and route management interface.

## Features

- Virtual scrolling spreadsheet for large delivery lists
- Driver assignment and time slot management
- Geographic clustering for route optimization
- Route export functionality
- Integration with cluster service for delivery grouping

## Components

- `DeliverySpreadsheet.tsx` - Main spreadsheet component
- `ClusterMap.tsx` - Map visualization of delivery clusters
- `components/` - Popups for driver assignment, time assignment, cluster generation
- `types/` - Delivery-specific type definitions
