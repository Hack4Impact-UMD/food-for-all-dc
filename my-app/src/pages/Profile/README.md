# Client Profile Page

Comprehensive client profile editor with multiple form sections for managing client information.

## Form Sections

- **BasicInfoForm** - Name, contact, address
- **DeliveryInfoForm** - Delivery preferences and scheduling
- **DietaryPreferencesForm** - Dietary restrictions and preferences
- **HealthConditionsForm** - Health-related information
- **DeliveryLogForm** - Delivery history management
- **MiscellaneousForm** - Additional notes and tags

## Components

- `Profile.tsx` - Main page orchestrator
- `components/` - Form components and shared form utilities
- `Tags/` - Tag management for client categorization
