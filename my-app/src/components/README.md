# Components

Reusable UI components used throughout the application.

## Structure

- **common/** - Shared components (Button, Input, Modal, Preloader)
- **Spreadsheet/** - Client spreadsheet with virtual scrolling
- **UsersSpreadsheet/** - User management spreadsheet
- **skeletons/** - Loading skeleton components
- **performance/** - Performance monitoring components

## Key Components

- `NotificationProvider` - Global notification system
- `VirtualScroll` - Virtual scrolling utility
- `ConfirmationModal` - Reusable confirmation dialogs
- `ErrorPopUp` - Error display component
