# Common Components

Shared, reusable UI components used across the application.

## Components

- **Button** - Standardized button component with variants
- **Input** - Text input, date input, and masked input fields
- **Modal** - Reusable modal dialog component
- **Preloader** - Loading indicator component

## Usage

Import from the common index:
```typescript
import { Button, Input, Modal } from '../common';
```

All components use CSS Modules for styling and follow the design system defined in `src/styles/theme.css`.
